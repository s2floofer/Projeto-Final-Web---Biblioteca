Sobre a Biblioteca

A bookTeca é um projeto desenvolvido para demonstrar o uso de tecnologias web modernas, incluindo HTML, CSS e JavaScript, com integração via Fetch API e armazenamento local.

Nosso objetivo é oferecer um ambiente simples, intuitivo e eficiente para o cadastro e gerenciamento de livros, simulando o funcionamento básico de um sistema real de biblioteca.

Objetivos do Projeto

  • Praticar requisições HTTP (GET, POST, PUT, DELETE)
  • Trabalhar com manipulação do DOM
  • Aplicar validações de formulário
  • Organizar código em arquivos separados
  • Desenvolver uma interface simples e responsiva

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Fetch API
  • LocalStorage

Este projeto faz parte de uma atividade acadêmica e demonstra a implementação de um CRUD completo em ambiente web.
